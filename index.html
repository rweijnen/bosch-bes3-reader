Bosch Smart System (BES3) Drive Unit Reader

Read-only diagnostic tool for Bosch Smart System eBikes. Connects to the drive unit's system controller over USB-C and reads identification/status data (serial number, product code, software versions, speed limits, etc). Runs entirely in your browser via WebUSB — nothing is sent to any server.

This tool only reads data. It never writes, configures, or tunes anything, and it never touches licensing/authorization systems. Requires Chrome, Edge, or another Chromium-based desktop browser (WebUSB is not available on iOS/Safari).
